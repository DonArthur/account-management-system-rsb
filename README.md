# Account Management System

A full-stack web application built with Spring Boot and React, featuring
JWT authentication, role-based authorization, user profile management,
and avatar uploads.

## Features
- JWT authentication & authorization
- Role-based access control
- User profile management
- Avatar upload
- Secure password handling
- Protected routes
- RESTful API

## Tech Stack

### Backend
- Java 17
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate
- PostgreSQL

### Frontend
- React
- React Router
- Axios
- Tailwind CSS
- Context API

## Architecture
- Controller / Service / Repository layers
- DTO separation
- Stateless JWT authentication
- Role-based security
